Uso de repositorios en GitHub¶

Resumen de la clase¶

En esta clase queremos aprender a usar los conceptos básicos de Git y al mismo timepo, crear un repositorio en GitHub para ilustrar las aplicaciones más básicas. Para eso utilizaremos Visual Studio Code (VSC), la terminal para conectarnos con Git, en conjunto con la función Source Control de VSC que está integrada y se conecta también con Git.
