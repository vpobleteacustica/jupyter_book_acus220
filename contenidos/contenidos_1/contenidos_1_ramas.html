2. Ramas en el repositorio.¶

  • Siempre al principio en el repositorio hay una sola rama que se llama main y que es la rama principal.

  • Trae ahora a tu máquina local, este repositorio GitHub recién creado.

  • Por simplicidad, en el escritorio de tu máquina local, crea una nueva carpeta. Por ejemplo, llámale github.

  • Cambia el directorio (cd) y entra a tu nueva carpeta. Abre tu terminal y activa tu ambiente virtual. Asegúrate de tener instalado python y git.

  • Clona el repositorio usando un cliente git.

  • Copia la dirección del repositorio que acabamos de crear en GitHub y en la terminal abierta, ejecuta la siguiente línea de comando:

kasparov@kasparov-MS-7B58:~/Escritorio/github$ git clone <link del repositorio GitHub>

2.1. Entra a la carpeta (cd).¶

  • En la terminal abierta, entra a la carpeta ejecutando la línea de comando:

kasparov@kasparov-MS-7B58:~/Escritorio/github$ cd uso_github

  • El único archivo que se ve es el README.md. Deberías ver lo siguiente:



  • El archivo .gitignore está oculto. Para ver los archivos ocultos ejecuta la siguiente línea de comando:

kasparov@kasparov-MS-7B58:~/Escritorio/github/uso_git$ ls -la

  • Si todo está bien, deberías ver:



  • También, observa la carpeta .git. Sin esta carpeta no podrías hacer, por ejemplo, lo siguiente:

kasparov@kasparov-MS-7B58:~/Escritorio/github/uso_git$ git status

  • Lo que te permite ver que esto es un proyecto Git. Siempre para ver el estado de tu repositorio, usa los comandos git status. Te muestra en qué rama estás y te dice que esta rama ya está actualizada.
